Python is a popular programming language.
Python can be used on a server to create web applications.
Created by Guido Van Rossum & released in 1991.
Used for:
    - web development (server side)
    - software development
    - mathematics
    - system scripting
    - create workflows
    - connect database systems
    - handle big data & perform complex mathematics
    - rapid prototyping, production ready software development

Works on different platforms (Windows, Mac, Linux, Raspberry Pi etc.)
Simple syntax
runs on an interpreter system
object-oriented or functional way
recent version is Python 3
Python designed for readability, uses few lines to complete a command,
    relies on indentation, using whitespace, to define scope such as scope of loops,
    functions and classes.

Python syntax can be executed directly by writing in the Command Line or by
    creating a python file on the server, using the .py file extension and running
    it in the Command Line.

Python indentation refers to the spaces at the beginning of a code line.
Python uses indentation to indicate a block of code.
variables are created when you assign a value to it
Python has no command for declaring a variable
comments start with a # in python
